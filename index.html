// 以下代码输出什么？
function* generator(i) {
    yield i;
    yield i * 2;
}
const gen = generator(10);
console.log(gen.next().value);  // 输出什么？
console.log(gen.next().value);  // 输出什么？

答案解析 
 
    # 一般的函数在执行之后是不能中途暂停的，但是，生成器函数却可以中途“停下”，之后可以再从停下的地方继续。
    # 首先，我们用 10 作为参数来初始化生成器函数，然后使用 next() 方法一步步执行生成器。
    # 第一次执行生成器的时候 i 的值为 10，遇到第一个 yield 关键字，它要生成 i 的值。此时，生成器“暂停”，生成了 10。
    # 然后，我们再执行 next() 方法，生成器会从刚才暂停的地方继续，这个时候 i 还是 10。
    # 于是我们走到了第二个 yield 关键字处，这时候需要生成的值是 i*2， i 为 10，那么此时生成的值便是 20。
    # 所以这道题的最终结果是 10,20。
